## 简单代码生成器

功能很简单，就是利用excel中的数据替换指定代码片段来生成脚本。具有三种模式，

1. 单元格模式：最简单的循环单元格的数据，请手动编辑单元格；
2. 单一脚本：就是所有的数据都是执行同一个脚本；
3. 多脚本：每一行执行的脚本可以不同，必须事先指定，这个灵活性最强。
